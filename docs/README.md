## 🔄 Como manter este guia atualizado

Este guia depende de ofertas externas que mudam com frequência. Para mantê-lo confiável:

1. **Fonte da verdade:** o [catálogo oficial do Pack](https://education.github.com/pack) e a página de cada parceiro. Confirme lá antes de alterar qualquer valor.
2. **Data única:** atualize a data em **um lugar** — o campo *Última atualização* do [README principal](../README.md).
3. **Marque o que não confirmou:** valores ou prazos sem reconfirmação recebem **(verificar)** até alguém checar na fonte.
4. **Prazos sensíveis:** datas de expiração (ex.: vouchers) vencem rápido — revise-as a cada poucos meses.
