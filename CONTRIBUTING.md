# ❤️ Contribuindo

Valeu por querer contribuir com este guia.

Se você encontrou um link quebrado, viu alguma informação desatualizada ou quer melhorar um trecho do texto, pode abrir uma issue ou enviar um pull request. Qualquer ajuda já faz diferença. 🚀

## ✨ Como ajudar

### 🐛 Issues
Abra uma **issue** se encontrar, por exemplo:

- link quebrado
- informação desatualizada em relação ao [GitHub Education](https://education.github.com/)
- erro de escrita
- trecho confuso ou que pode ficar melhor

Se puder, já diga exatamente onde está o problema. Isso ajuda bastante na revisão.

### 🔀 Pull requests
Se quiser corrigir direto, pode abrir um **pull request**.

De preferência:
- explique rapidamente o que mudou
- mantenha o PR focado em uma coisa só
- revise links, texto e formatação antes de enviar
