# Guia definitivo de benefícios e recursos gratuitos (FIAP + GitHub Student Pack) para iniciantes em programação 🧭

- Escrever código simples sem "colar" tudo;
- Entender erros básicos com segurança;
- Montar um projeto pequeno com autonomia.

### 2) Regra prática: 1–1–1 ✅

Escolha 1 item em cada grupo:

- **1 fonte principal de estudo** → Curso em Vídeo, Code.education ou Alura  
- **1 lugar para praticar** → VS Code (ou editor equivalente)  
- **1 forma de organizar** → Notion, caderno ou Google Docs

Todo o resto é opcional no começo. 🎯

### 3) Heurística de decisão (objetiva)

- Se você **não entende nada e sente vergonha**: comece com vídeo (Curso em Vídeo) e mantenha o foco na faculdade.
- Se você **entende a explicação, mas não consegue fazer sozinho**: pratique no Code.education / Codédex com o GitHub Student Pack.
- Se a faculdade está difícil por **falta de base**: faça o Nivelamento antes de se comparar com o ritmo dos outros.
- Se você quer **resultado visível mais rápido**: avance por etapas; primeiro funciona, depois melhora.
- Se você quer **IA ajudando**: use a IA como explicadora e revisora, não como substituta do seu esforço.

## O que priorizar agora, o que deixar para depois

### 🔥 Prioridade máxima (iniciante absoluto)

- FIAP + Alura (se você tem acesso)
- Nivelamento FIAP ON quando faltar base
- GitHub Student Developer Pack (alto impacto agora)

### ⏭️ Útil em seguida

- Educative / Frontend Masters / Scrimba
- Notion para organizar estudos e entregas
- Talent Lab para organização, ansiedade e carreira
- GitHub Skills para praticar versionamento

### 🧩 Importante mais para depois

- Azure / DigitalOcean / Heroku
- MongoDB Atlas
- Domínio (Namecheap / .TECH) — pode sair grátis com o GitHub Student Pack

### 🧪 Legal, mas não essencial agora

- Certificações muito cedo
- Repositórios avançados de arquitetura

---

## Minha dica final 🎯

Se você é iniciante, o melhor uso dos benefícios é:

1. Foque em aprender e praticar até construir um projeto pequeno sozinho.
2. Só depois use cloud, hosting e créditos — isso acelera quando a base já está pronta.
3. Progresso vem com consistência, não com pressa. 🚀
